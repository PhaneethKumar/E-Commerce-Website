E-Commerce Website

This project is a basic front-end web application that demonstrates a simple authentication system along with a product listing interface. It includes user signup and login functionality with form validation, as well as a responsive product page that displays items in a structured grid layout.

User data is stored locally using the browser’s Local Storage. After a successful login, the user session is managed using Session Storage and is automatically cleared when the session ends. Direct access to the product page via its URL is restricted by validating the loggedInUser key in Session Storage, ensuring that only authenticated users can access it.

The application focuses on clean UI design, a modular structure, and fundamental web development concepts. This project was built as a trial task to showcase my approach to problem-solving, code organization, and front-end development practices.
