# E-Commerce-Website

This project is a basic front-end web application that demonstrates a simple authentication system and a product listing interface. It includes user signup and login functionality with form validation, along with a responsive product page displaying items in a structured grid layout.  User data is stored locally using browser storage, and the application focuses on clean UI design, modular structure, and fundamental web development concepts. This project was built as a trial task to showcase my approach to problem-solving, code organization, and front-end development practices.
